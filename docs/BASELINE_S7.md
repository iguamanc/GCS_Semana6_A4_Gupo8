# Baseline Semana 7 – ParqueAndo

## Versión
v0.3.0

## Fecha
19-02-2026

## Objetivo
Establecer línea base verificable para práctica de Gestión de Configuración (GCS),
incluyendo evidencia de control de cambios, trazabilidad y tablero Kanban.

## Elementos de Configuración (CIs)

- Código fuente: /src
- Documentación: /docs
- Configuración: /config
- Plantillas PR: /.github
- Tests: /tests

## Reglas de control

- No commits directos a main.
- Todo cambio debe tener Issue (CHG-XX).
- Todo cambio debe pasar por Pull Request aprobado.
- Toda versión debe tener Tag y Release.
